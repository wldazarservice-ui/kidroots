# KidRoots 🌍

Application educative pour les enfants de 4 a 7 ans.
Decouvre les pays du monde a travers des jeux !

## Demarrage rapide

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Stack
- React 18 + Vite
- Tailwind CSS
- PWA ready (manifest.json)

## Deploiement
Connecter le repo GitHub a Netlify.
Build command: npm run build
Publish directory: dist
